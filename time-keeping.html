EffyTools is a system of small and smart services for personal information management (collecting, measuring, reviewing). This is an efficiency management kit which is build keeping in mind the personal experience of the creator, his habits and techniques he studied in the area of personal productivity, time management, getting projects done, information management (blog). All the services are pretty small and low-level to be involved in much bigger practices, like Inbox or Urgent-vs-Important matrix.
The data are stored inside user's favourite browser (via localStorage technology). This is a basic scenario. But also the data can be downloaded to local files or stored on the server (terms of use).
You are welcome! Please, look at service you like, explore its interface to see the functionality. It is made to be easy in use. Features like tags, overall search, pages, settings, custom clipboard buffer, wiki syntax support are also present. There are a few short descriptions below.
Service is for storing notes of any kind, for example, diary notes. Constructed automatically the notes line contains notes ordered by time. Tags and text search features facilitate the process of review.
Service is for measuring task duration, getting statistics. This is a time keeping tool. It helps to see the big picture and manage the tasks. Tasks are divided into three type: templates, tasks in progress, finished ones. Statistics report is generated by information about finished tasks.
Service is for keeping information about events. One can review the calandar, pick a date and see the events he stored before for that day. Events are ordered within a day, so each of them can represent the task or the note binded to a particular time. Also there are upcoming events and historic ones shown.
Service is for creating simple diagrams with blocks and arrows. It is easy to put any text block at any place inside a window, then put another one and connect these two with an arrow. There are two modes: rearranging blocks by dragging and dropping; connecting blocks with arrows by means of mouse.
Service is for creating lists. Lists can contain tasks, wishes, products to buy or any other items. It also can be used during brain storming. Items inside a list can be sorted or rearranged. When new list is created its name is a text of the first element. One item or group can be moved form one list to another.

EffyTools это система небольших сервисов для работы с личной информацией (планирование, учёт, хранение, измерение времени). Это инструменты управления личной эффективностью, которые были построены на основе опыта Автора, его привычек и освоенных им методологий и практик по тайм-менеджменту, управлению делами и информацией (блог). Сервисы носят базовый характер и могут быть использованы как фундамент для более комплексных систем и техник (например, "Входящие", матрица "Срочно-Важно" и другие).
Основной сценарий работы подразумевает, что все данные пользователя хранятся внутри его любимого браузера (технология localStorage), но также есть возможность сохранять данные в локальные файлы, или хранить пакеты данных на сервере (условия использования).
Пользователю предлагается самостоятельно исследовать интерфейс сервисов для подробного ознакомления с функциями и преимуществами каждого. Часть функций имеет общий характер: система тегов, страницы, поиск по всем данным, общие настройки, буфер обмена артефактами, викификация текста. Ниже подробнее представлены некоторые особенности сервисов.
Основное назначение: сохранение заметок, ведение личного дневника. Используется для запоминания информации, обзора произошедшего ранее. Текстовые сообщения сохраняются в виде ленты, упорядоченные по времени. Система тегов и текстового поиска позволяют найти внесённую когда-то информацию.
Основное назначение: проведение хронометража. Используется для измерения времени выполнения задачи, общего времени, оценки объема выполненного и соотношения задач. Задачи делятся на шаблоны (порождают активные), активные (те, которые сейчас выполняются) и выполненные (законченные). На основе информации о выполненных задачах автоматически генерируется статистика (общая и по дням).
Основное назначение: ежедневник, обзорный календарь. Используется для хранения информации о событиях и важных датах, составления плана дня и обзора истории. Панель с месяцами и датами даёт общее представление о течении времени. Ниже отображаются события или задачи на выбраный день (по-умолчанию, это текущий день), также, что будет далее (завтра и в последующие дни), и что было до (история).
Основное назначение: быстрое планирование, карты памяти, схемы. Используется для рисования схем с фрагментами текста и стрелками между ними, с возможностью простой перегруппировки. Фрагменты могут быть занесены в произвольном порядке. Есть два режима: перемещение фрагментов с помощью мыши; рисование стрелок (мышь) и копирование текста.
Основное назначение: ведение произвольных списков, гибкое планирование. Используется для составления списков дел, покупок, заметок, для записей при мозговых штурмах, с последующей сортировкой по приоритетам или другим критериям. Список создаётся по имени своего первого элемента. Есть возможность перемещения пунктов между списками, групповые операции (сортировка, перемещение).
